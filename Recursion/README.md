##### Exercise 8: 
    Xếp n quân hậu trên một bàn cờ quốc tế
    sao cho không có 2 quân hậu nào ăn được
    nhau
    Mô hình hoá:
    x[1, . . ., n] trong đó x[i] là hàng của
    quân hậu xếp trên cột i, với mọi i = 1, …, n
    
    Thuộc tính P:
    x[i] != x[j], với mọi 1 <= i < j <= n
    x[i] +i != x[j] +j, với mọi 1 <= i < j <= n
    x[i] –i != x[j] -j, với mọi 1 <= i < j <= n
